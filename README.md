# react-native-migration-audit# React Native Migration Audit

CLI tool for auditing React Native and Expo repositories and generating migration risk reports.

## Features

- React Native / Expo version analysis
- Dependency analysis
- Native tooling analysis
- Deprecated API detection
- Migration-sensitive area classification
- Complexity scoring
- Upgrade recommendations
- Proposal generation

## Usage

```bash
yarn install

yarn audit /path/to/react-native-project
